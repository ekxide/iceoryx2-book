# iceoryx2-book

The `iceoryx2` book.

## Setup

1. Install `poetry` (if not already installed):

    ```console
    curl -sSL https://install.python-poetry.org | python3 -
    ```

1. Install dependencies

    ```console
    cd iceoryx2-book/
    make install
    ```

## Development

### Static Build

```console
cd iceoryx2-book/
make html
```

### Development Server

```console
cd iceoryx2-book/
make serve
```
